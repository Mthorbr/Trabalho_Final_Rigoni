O Projeto KickStream

A KickStream é uma plataforma de catálogo de jogos de futebol desenvolvida como projeto integrador de infraestrutura em nuvem. O objetivo central é demonstrar a aplicação prática de conceitos de Big Data e computação elástica.

Toda a infraestrutura está hospedada na Amazon Web Services (AWS), utilizando recursos como Virtual Private Cloud (VPC) para segurança de rede, Application Load Balancer (ALB) e Auto Scaling Group (ASG) para garantir alta disponibilidade, e o Amazon DynamoDB como camada NoSQL para persistência dos dados.

O player de vídeo presente no portal tem a finalidade de "mockar" (simular) uma transmissão real, concentrando o foco analítico da disciplina na robustez e escalabilidade do cluster de servidores no back-end.
